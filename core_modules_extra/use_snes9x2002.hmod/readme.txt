=== Use SNES9x2002 instead of SNES9x2010 ===

This module will patch /bin/snes file to use SNES9x2002 instead of SNES9x2010.
Make sure that snes9x2002 module is installed before installing this.

Available executables and arguments:
- /bin/snes <rom> <clover_args>
  runs "snes9x2002" core
- /bin/snes02 <rom> <clover_args>
  runs "snes9x2002" core
