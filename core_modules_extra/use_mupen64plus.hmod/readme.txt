=== Use Mupen64Plus instead of GLupeN64 ===

This module will patch /bin/n64 file to use Mupen64Plus instead of GLupeN64.
Make sure that mupen64plus module is installed before installing this.

Available executables and arguments:
- /bin/n64 <rom> <clover_args>
  runs "mupen64plus" core
- /bin/n64p <rom> <clover_args>
  runs "mupen64plus" core
